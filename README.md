# Caesar Cipher Tool

Программа для шифрования и расшифровки текста с использованием шифра Цезаря. Поддерживает работу с большими файлами, валидацию входных данных, а также опциональные режимы brute force и статистического анализа.

## Возможности

- Шифрование и расшифровка текста с заданным ключом (сдвигом).
- Работа с большими текстовыми файлами (чтение/запись).
- Валидация входных данных (существование файлов, допустимость ключей).
- Режимы работы:
  - Шифрование текста.
  - Расшифровка с известным ключом.
  - Расшифровка методом brute force (перебор всех ключей).
  - Расшифровка методом статистического анализа (автоматический подбор ключа).
- Интерфейс: текстовое меню или опционально графический интерфейс.
- Обработка ошибок и исключений.
- Оптимизация для производительности.
- Для разработки используйте JDK не ниже 21

## Ход работы

1. Сделайте Fork этого репозитория в свой аккаунт
2. Клонируйте репозиторий из своего аккаунта через ссылку (в своем аккаунте сделайте code -> local -> HTTPS -> copy URL)
3. В Idea создайте новый проект нажав Get from VCS -> вставте скопированую на предыдущем этапе ссылку
4. После создания проекта в корневой директории cryptanalyzer создайте директорию java и пометьте ее как директорию для исходников (Mark Dyrectory as -> Sources root), папка должна стать синего цвета
5. Создавайте свои классы в папке java
6. Не забывайте делать add когда добавили новый класс и делать commit для сохранения работающего кода
7. Погда выполнили проект необходимо сделать push в свой репозиторий на github
8. После выполнения проекта и загрузки его в свой репозиторий необходимо заполнить форму для сдачи проекта
